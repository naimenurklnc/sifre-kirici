#  Python Şifre Kırıcı

Threading ile hızlandırılmış, wordlist tabanlı brute force şifre kırıcı.

## Özellikler
- Wordlist'ten şifreleri sırayla dener
- Threading ile yüksek hız
- Şifre bulununca otomatik durur
- Tarama başlangıç/bitiş süresi gösterir

## Kullanım
```bash
py sifre_kirici.py
```
## Kullanılan Teknolojiler
- Python 3
- threading modülü
- time modülü

##  Yasal Uyarı
Bu araç yalnızca eğitim amaçlı ve izin verilen sistemlerde kullanılmalıdır.
